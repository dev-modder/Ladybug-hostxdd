Deployment Guide

1

Create Your Bot

Fill out the form above with your bot configuration. You'll need a token from your chosen platform.

2

Deploy to Render

Click deploy and your bot will be hosted on Render's free tier automatically.

# Your bot will be deployed to: https://your-bot.onrender.com
3

Set Environment Variables

Configure these environment variables in your Render dashboard:

BOT_TOKEN=your_token_here
NODE_ENV=production
PORT=3000
4

Test Your Bot

Visit your bot's health endpoint to verify it's running:

curl https://your-bot.onrender.com/health

Quick Start Code

// index.js - Minimal bot setup const { LadybugMiniBot, quickDeploy } = require('ladybug-mini-bot'); // Create your bot const bot = new LadybugMiniBot({ name: 'My Bot', prefix: '!', owner: '@yourname' }); // Add a custom command bot.command('hello', async (ctx) => { return ctx.reply('Hello World! 👋'); }, { description: 'Say hello' }); // Deploy! quickDeploy(bot.config);
